  <files>frontend/vite.config.ts, frontend/src/index.css, frontend/tsconfig.json, frontend/tsconfig.app.json, frontend/components.json, frontend/package.json</files>
  <action>
Tailwind CSS v4와 shadcn/ui를 설정한다. 반드시 v4 방식(CSS 기반)을 따를 것. v3 방식(JS config) 사용 금지.

1. Tailwind v4 설치 (frontend/ 디렉토리에서):
   - `npm install tailwindcss @tailwindcss/vite`

2. vite.config.ts 수정:
   - `import tailwindcss from '@tailwindcss/vite'` 추가
   - `import path from 'path'` 추가
   - plugins 배열에 `tailwindcss()` 추가
   - resolve.alias에 `'@': path.resolve(__dirname, './src')` 추가
   - 01-RESEARCH.md Pattern 4 참조

3. frontend/src/index.css 수정:
   - 기존 내용을 `@import "tailwindcss";`로 교체
   - 안티패턴: `@tailwind base; @tailwind components; @tailwind utilities;` 사용 금지 (v3 문법)
   - tailwind.config.js 파일 생성 금지 (v4는 CSS 기반 설정)

4. tsconfig.json 수정:
   - compilerOptions에 `"baseUrl": "."` 추가
   - compilerOptions에 `"paths": { "@/*": ["./src/*"] }` 추가

5. tsconfig.app.json 수정:
   - compilerOptions에 `"baseUrl": "."` 추가
   - compilerOptions에 `"paths": { "@/*": ["./src/*"] }` 추가

6. shadcn/ui 초기화 (frontend/ 디렉토리에서):
   - `npx shadcn@latest init -d` 실행 (-d 플래그로 기본 설정 사용)
   - components.json 생성 확인
   - 만약 대화형 프롬프트가 나오면: TypeScript yes, Style Default, Base color Slate, CSS variables yes

안티패턴:
- tailwind.config.js 생성 금지 (v4)
- postcss.config.js 별도 생성 금지 (v4에 내장)
- @tailwind 디렉티브 사용 금지 (v3 문법)
