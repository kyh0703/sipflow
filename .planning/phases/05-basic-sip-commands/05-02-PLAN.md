**Step 2: Implement MakeCall**

Follow the pattern from RESEARCH Pattern 3 exactly:
```go
func (s *SIPService) MakeCall(nodeID, targetURI string) Response[string]
```
- Guard: check s.entClient != nil (NO_PROJECT error)
- Guard: check s.sessionManager.HasActiveCall(nodeID) -- reject with "CALL_ACTIVE" if true (one-call-per-UA)
- Get diago via s.uaManager.GetDiago(nodeID) -- fail with "UA_NOT_FOUND" if error
- Parse target URI with sip.ParseUri("sip:" + targetURI) -- fail with "INVALID_URI" if error
- Generate callID with fmt.Sprintf("%s-%d", nodeID, time.Now().UnixNano()) (simple, no uuid dependency needed)
- Create cancellable context: ctx, cancel := context.WithCancel(context.Background())
- Create ActiveSession with cancel, nodeID, state=CallStateDialing, nil dialog
- Add to sessionManager
- Launch goroutine that:
  - defer s.sessionManager.Remove(callID)
  - Creates InviteOptions with OnResponse callback that emits "sip:callState" events for provisional responses (180=ringing, other 1xx=progress)
  - Calls dg.Invite(ctx, recipient, opts)
  - On error: emit "sip:callState" failed event with sipgo.ErrDialogResponse extraction (errors.As pattern from research)
  - On success: set session.Dialog, set state=Established, emit "sip:callState" established
  - Wait on dialog.Context().Done()
  - Emit "sip:callState" terminated
  - defer dialog.Close()
- Return Success(callID) immediately

**Step 3: Implement Bye**

Follow RESEARCH Pattern 4:
```go
func (s *SIPService) Bye(callID string) Response[bool]
```
- Get session from sessionManager -- "SESSION_NOT_FOUND" if missing
- Check state == CallStateEstablished -- "INVALID_STATE" if not
- Check session.Dialog != nil -- "NO_DIALOG" if nil
- Call session.Dialog.Hangup(ctx) with 5s timeout
- Return Success(true)

**Step 4: Implement Cancel**

Follow RESEARCH Pattern 5:
```go
func (s *SIPService) Cancel(callID string) Response[bool]
```
- Get session from sessionManager -- "SESSION_NOT_FOUND" if missing
- Check state == CallStateDialing || CallStateRinging -- "INVALID_STATE" otherwise
- Call session.Cancel() (context cancellation triggers SIP CANCEL automatically)
- Return Success(true)

**Imports needed in sip_service.go:** Add "errors", "github.com/emiago/diago", "github.com/emiago/sipgo", "time" (some may already exist).

**Important:** Do NOT add `"github.com/google/uuid"` -- use simple callID generation instead.
