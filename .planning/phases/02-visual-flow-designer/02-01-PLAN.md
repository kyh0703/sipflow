       - Use Tailwind CSS classes only

    5. Create `CommandNode.tsx`:
       - Import NodeProps from @xyflow/react, icons from commandIcons.ts
       - Look up icon from commandIcons map using data.command
       - Render: rounded-lg border-2 card with dynamic icon + label
       - Handle target on Left, Handle source on Right
       - Selected state: border-primary + shadow-lg; default: border-orange-300 + orange-50 bg
       - Fallback icon if command not found

    6. Create `EventNode.tsx`:
       - Timer icon from lucide-react
       - Render: rounded-lg border-2 card with Timer icon + label
       - Handle target on Left, Handle source on Right
       - Selected state: border-primary + shadow-lg; default: border-green-300 + green-50 bg

    7. Create `frontend/src/components/flow/nodes/index.ts`:

       - Import each node component and wrap with React.memo()
       - Export const nodeTypes = { sipInstance, command, event } satisfies NodeTypes
