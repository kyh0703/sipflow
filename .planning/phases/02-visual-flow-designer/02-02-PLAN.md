    1. Create `frontend/src/components/flow/FlowCanvas.tsx`:
       - Import ReactFlow, Background, Controls, MiniMap, Panel, useReactFlow from @xyflow/react
       - Import '@xyflow/react/dist/style.css'
       - Import nodeTypes from './nodes' and edgeTypes from './edges'
       - Import useFlowStore with selectors (NOT full store access)
       - Use selectors: `useFlowStore(s => s.nodes)`, `useFlowStore(s => s.edges)`, `useFlowStore(s => s.actions)`
       - Implement onDragOver handler: preventDefault, set dropEffect 'move'
       - Implement onDrop handler:
         - Get nodeType from `event.dataTransfer.getData('application/xyflow')`
         - If empty, return
         - Use `screenToFlowPosition({ x: event.clientX, y: event.clientY })` for correct position
         - Parse nodeType: split on ':' to get [type, subtype]
         - Create node with unique id (`node-${Date.now()}`), type, position, and data based on subtype
         - For sipInstance: data = { label: 'SIP UA' }
         - For command: data = { label: subtype display name, command: subtype }
         - For event: data = { label: 'Wait Event', eventType: 'wait' }
         - Call actions.addNode(newNode)
       - Implement onNodeClick: call actions.setSelectedNode(node.id)
       - Implement onPaneClick: call actions.setSelectedNode(null)
       - Wire onNodesChange, onEdgesChange, onConnect from store actions
       - Pass isValidConnection that blocks only self-connections (return source !== target)
       - ReactFlow props: fitView, minZoom={0.1}, maxZoom={3}, fitViewOptions={{ padding: 0.2 }}
       - Include: <Background />, <Controls />, <MiniMap zoomable pannable />
       - Show empty canvas message when nodes.length === 0: centered text "Drag nodes from the sidebar to build your SIP flow"
       - Wrap all callbacks in useCallback

    2. Create `frontend/src/components/flow/LeftSidebar.tsx`:
       - Install shadcn/ui Accordion component if not already:
         ```bash
         cd frontend && npx shadcn@latest add accordion
         ```
       - Define nodeCategories array with 3 groups:
         - SIP Instance: [{ type: 'sipInstance', label: 'SIP UA', command: 'sipInstance' }]
         - Commands: [makeCall, hold, retrieve, blindTransfer, muteTransfer, bye, cancel, busy] — each with type: 'command', label, and command key
         - Events: [{ type: 'event', label: 'Wait Event', command: 'wait' }]
       - Use Accordion from shadcn/ui with type="multiple" defaultValue={['SIP Instance', 'Commands', 'Events']} (all expanded by default)
       - Each item is draggable div with onDragStart handler:
         - For command items: `dataTransfer.setData('application/xyflow', 'command:makeCall')`
         - For sipInstance: `dataTransfer.setData('application/xyflow', 'sipInstance')`
         - For event: `dataTransfer.setData('application/xyflow', 'event:wait')`
         - Set effectAllowed = 'move'
       - Each item shows lucide icon (from commandIcons mapping) + label text
       - Grid layout: 2 columns for items within each group
       - Style: cursor-grab, hover:bg-muted, active:cursor-grabbing, border-dashed rounded cards
       - Read sidebarOpen from useFlowStore, return null if !sidebarOpen
       - Width: w-60, border-r, bg-background, h-full, overflow-y-auto
