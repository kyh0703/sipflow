<task type="auto">
  <name>Task 1: Create FlowCanvas with ReactFlow and DnD drop handling</name>
  <files>
    frontend/src/components/flow/FlowCanvas.tsx
  </files>
  <action>
    Create `frontend/src/components/flow/FlowCanvas.tsx`:

    1. Import ReactFlow, Background, Controls, MiniMap, Panel, useReactFlow from @xyflow/react
    2. Import '@xyflow/react/dist/style.css'
    3. Import nodeTypes from './nodes' and edgeTypes from './edges'
    4. Import useFlowStore with selectors (NOT full store access):
       - `useFlowStore(s => s.nodes)`, `useFlowStore(s => s.edges)`, `useFlowStore(s => s.actions)`
    5. Implement onDragOver handler: preventDefault, set dropEffect 'move'
    6. Implement onDrop handler:
       - Get nodeType from `event.dataTransfer.getData('application/xyflow')`
       - If empty, return
       - Use `screenToFlowPosition({ x: event.clientX, y: event.clientY })` for correct position
       - Parse nodeType: split on ':' to get [type, subtype]
       - Create node with unique id (`node-${Date.now()}`), type, position, and data based on subtype
       - For sipInstance: data = { label: 'SIP UA' }
       - For command: data = { label: subtype display name, command: subtype }
       - For event: data = { label: 'Wait Event', eventType: 'wait' }
       - Call actions.addNode(newNode)
    7. Implement onNodeClick: call actions.setSelectedNode(node.id)
    8. Implement onPaneClick: call actions.setSelectedNode(null)
    9. Wire onNodesChange, onEdgesChange, onConnect from store actions
    10. Pass isValidConnection that blocks only self-connections (return source !== target)
    11. ReactFlow props: fitView, minZoom={0.1}, maxZoom={3}, fitViewOptions={{ padding: 0.2 }}
    12. Include: <Background />, <Controls />, <MiniMap zoomable pannable />
    13. Show empty canvas message when nodes.length === 0: centered text "Drag nodes from the sidebar to build your SIP flow"
    14. Wrap all callbacks in useCallback
  </action>
  <verify>

    TypeScript compiles. FlowCanvas has onDrop with screenToFlowPosition.
  </verify>

    - Drag-and-drop from sidebar creates nodes at correct canvas position
    - Node click selects node, pane click deselects
    - Edge connections work via handle dragging
    - Empty canvas shows placeholder message
  </done>
